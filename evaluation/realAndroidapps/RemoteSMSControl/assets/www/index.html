Besides all functions the free version SMS Remote Control GSM has, the following features are available in de PRO-version:
  • A possibility to protect the APP with a pin code
  • This pin code can be transferred to the SMS by using #### in the message
  • When you open an account at Clickatel, you can send SMS via the internet and you don't need a sim card or a phone, a tablet will do
  • No ads are shown
  • It is possible to add the name and phone number of your alarm company, and call them with a click from the start screen
  • The XML-parameter file is not readable so your codes and commands are kept secret
